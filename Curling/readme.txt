Voici les touches utilisés pour le jeux:

********* POUR LE CLAVIER ***********

- C : Bascule de type de camera
- Shift : Bascule de type de spin
- Space : Passer au prochain lancer lorsque le message s'affiche

********* POUR LA SOURIS ***********

- Clic gauche: Passer au prochain lancer lorsque le message s'affiche
- Clic gauche (prolongé) : Augmenter la vitesse de lancer selon le temps de maintien du clic
